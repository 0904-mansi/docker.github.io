Quick-Start

  1. Start Launchpad or open /Applications/Docker.
  2. Open the Docker Quickstart Terminal App.
  3. At the command line, test your installation by entering:
    docker run hello-world
  4. Visit the Docker OS X installation to learn more about your new installation.
